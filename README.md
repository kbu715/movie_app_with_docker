# To use this application,

1. docker 실행
2. server -> config -> dev.js MongoDB 설정
3. server -> config -> awsconfig.json 설정
4. docker-compose up --build 실행

7. Node.js

9. Ant Design & Material UI
10. docker & Travis CI
